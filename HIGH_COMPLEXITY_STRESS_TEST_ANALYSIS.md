## üîç ROOT CAUSE ANALYSIS

### **Why Did the Model Struggle with High-Complexity Queries?**

1. **Training Data Limitations**
   - Model was trained on 7,888 general college admissions examples
   - **Lacks current 2025 policy data** (FAFSA Simplification Act, new SAI calculations)
   - **Lacks technical policy documents** (ASSIST articulations, NCAA manuals, USCIS guidance)
   - **Lacks institutional-specific data** (school-by-school aid policies, transfer requirements)

2. **No Real-Time Data Access**
   - Cannot access actual URLs or current websites
   - Cannot pull live data from StudentAid.gov, ASSIST.org, Common Data Sets
   - Cannot verify current policies or numbers

3. **Hallucination Risk**
   - When asked for specific data it doesn't have, model fabricates plausible-sounding numbers
   - Examples: CS 105/106 prerequisites, selectivity percentages, TCO calculations
   - **Critical Issue:** Fabricated data is dangerous for college counseling

4. **Insufficient Depth for Technical Queries**
   - Model performs well on general advice (essay writing, personalized counseling)
   - **Struggles with technical requirements** (ASSIST course mapping, NCAA rules, visa regulations)
   - **Lacks structured output capabilities** (tables, decision trees, Gantt charts)
